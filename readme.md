# JP-Tech-Job-Hunting-Filter
新卒就活生の不安を煽るアフィリエイトサイト・低品質メディアをブロックするuBlock Origin用フィルターリストです。
IT系の新卒就活をするうえで、不要な情報が氾濫する昨今の検索結果により、義憤に駆られてこのフィルターを作成しました。
比較的自社記事が少ないという理由と、現実的な就職活動が難しくなるという理由からマイナビなどのような求人掲載サイトはこのリストから除外しています。

## 購読方法
uBlock Originをインストール後、以下のURLを「フィルターリストを購読」に追加してください。
https://raw.githubusercontent.com/HK416-Ger/JP-Tech-Job-Hunting-Filter/main/filter.txt

## 対象サイト
- アフィリエイト目的の無名就活エージェントサイト
- AI生成の仮想体験談を掲載するサイト
- ファクトチェックを名乗る低品質メディア
- 就活不安を煽るブログ
- 大手エージェント系サイト（ただし、マイナビやリクナビのような求人掲載サイトはリストから除外）
